# Personal Portfolio

This is a modern developer portfolio built with Next.js and TypeScript. It showcases projects, skills, experience, and more in a visually appealing and interactive way.

## Features

-   **Next.js 14**: Fast, SEO-friendly React framework
-   **TypeScript**: Type-safe codebase
-   **Component-based**: Modular and reusable components
-   **Responsive Design**: Mobile-friendly layout
-   **Smooth Scrolling**: Enhanced user experience
-   **Dark/Light Theme Toggle**
-   **Project, Skills, Reviews, and More**: Easily customizable sections

## Folder Structure

```
components/         # UI and page components
components/shared/  # Shared UI elements
components/sub/     # Sub-components for sections
components/ui/      # UI primitives (button, dropdown, etc.)
lib/                # Providers and utilities
public/             # Static assets (images, data)
app/                # Next.js app directory (pages, layout)
```

## Getting Started

1. **Clone the repository:**
    ```bash
    git clone <repo-url>
    cd dev-portfolio
    ```
2. **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```
3. **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```
4. Open [http://localhost:3000](http://localhost:3000) to view your portfolio.

## Customization

-   Update content in the `components/` and `public/` folders.
-   Edit `app/page.tsx` and other files to personalize sections.
-   Add your own images and data in `public/projects/`, `public/skills/`, etc.

## Scripts

-   `npm run dev` — Start development server
-   `npm run build` — Build for production
-   `npm run start` — Start production server
-   `npm run lint` — Lint code
